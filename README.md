Task 3: AI-CHATBOT-WITH-NLP (spaCy)

Introduction

This project is developed as part of Task-3 of the CODTECH Python Programming Internship.
The objective of this task is to build a simple Natural Language Processing (NLP) based chatbot using Python.
The chatbot can understand user input, detect intent using semantic similarity, and respond appropriately using predefined responses.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Objective

The main objectives of this task are:

- To understand the basics of Natural Language Processing (NLP)
- To use the spaCy library for text processing and similarity matching
- To design a rule-based chatbot using intent detection
- To allow users to interact with the chatbot through the command line
- To simulate human-like conversation using predefined intents and responses 

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Implementation Details

The chatbot is implemented using Python with the following components:
Libraries Used
- spaCy – For NLP processing and semantic similarity
- random – To generate varied responses

Key Implementation Steps
1. Load the pre-trained spaCy model en_core_web_md.
2. Define intents with:
- Keywords
- Corresponding responses
3. Process user input using spaCy NLP pipeline.
4. Detect intent based on similarity between user input and intent keywords.
5. Select a random response from the matched intent.
6. Continue conversation until the user types bye or exit.

Intent Categories
- Greeting
- Bot Identity (Name)
- Purpose
- Goodbye

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Output and Results

- The chatbot successfully responds to user inputs such as greetings, purpose-related questions, and exit commands.

- It dynamically selects responses based on detected intent.

- The chatbot runs in a continuous loop until the user exits.
